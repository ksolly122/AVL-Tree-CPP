# AVL Tree in C++

## Overview
This project is a C++ implementation of an AVL tree, which is a self-balancing binary search tree. It demonstrates low level data structure logic such as node management, recursive insertion, height tracking, balance factor checks, and tree rotations.

## What It Does
- Stores integer values in an AVL tree
- Automatically rebalances the tree after insertions
- Uses tree rotations to maintain balance
- Tracks node heights and balance factors
- Includes traversal and supporting tree operations

## Skills Demonstrated
- C++
- Data structures
- Binary search trees
- AVL tree balancing
- Recursion
- Pointer-based node management

## Why This Project Matters
AVL trees are a classic computer science data structure because they combine fast searching with self-balancing behavior. This project shows my understanding of rotations, recursion, and dynamic tree operations in C++.

## File Structure
- `avl_tree.cpp` - main AVL tree implementation

## How to Compile
```bash
g++ -std=c++11 avl_tree.cpp -o avl_tree
```

## How to Run
```bash
./avl_tree
```
